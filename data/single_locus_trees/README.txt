This directory contains the single locus tree files used in various analyses. 

The strongylocentrotidae/ directory contains the single locus tree file used in the concordance factor analysis. 

The s_clade directory contains the single locus tree files used in the PhyloNet analysis. The single_locus_trees_boot.nwk file has 100 boostrap trees per locus, while the single_locus_trees.nwk file contains just one tree string per locus. 

The delta/ directory contains the single locus tree files used in the delta analyses.
