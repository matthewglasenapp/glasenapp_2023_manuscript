This directory contains the results of the concordance factor analysis. The input/ directory contains the input files used to calculate concordance factor statistics. The output/ directory contains the raw output from running iqtree --gcf. The gcf_chisq/ directory contains a script for testing for significant differences in the counts of discordant NNI trees for each branch in the species tree. 

In the files within this directory, individuals are referred to by their sample name on NCBI. The following key links sample name to species name and SRA accession:
