The script generate_mt_tree_from_fasta.py takes an unaligned fasta file of mitochondrial genome assemblies, aligns them using Clustal, and estimates a phylogeny using IQ-TREE2. 

The file raw_mitochondrial_sequences.fasta is a fasta file of all unaligned mitochondrial genome assemblies available on NCBI for the strongylocentrotid urchin family.
