The script get_unique_exons_bed.sh gets all unique, protein-coding exons from a gff3 file and converts to bed using covert2bed (https://bedops.readthedocs.io/en/latest/content/reference/file-management/conversion/convert2bed.html). 

The script get_protein_coding_genes_gff.sh gets a file of all protein-coding gene records from a gff3 file in gff3 format. 

The script get_all_exons_sorted.sh gets all coding exons from a gff file and converts to bed format. The bed file is sorted by exon name to keep the hierarchical structure of gff3.

The script gff_to_bed.sh converts a gff3 file to bed file format using BEDOPS convert2bed.
